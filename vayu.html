Vayu- An interactive installation

Interactive installation at India HCI 2014, IIT Delhi





About the project

Vayu is a performative interactive installation which was showcased in IndiaHCI 2014, held at IIT Delhi. It is an arduino based installation with 15 pinwheels powered by 4 motor controllers. The installation is triggered with human action of blowing air. Blowing air in a microphone triggers pinwheels. The speed of rotation depends on intensity of blow. The installation was created to add some playfulness for participants, after long informative conference sessions.
